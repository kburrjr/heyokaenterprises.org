## Deploy to GitHub Pages

1. Create a new repo: `https://github.com/<your-username>/heyoka-enterprises`
2. Push these files to `main`
3. Go to **Settings → Pages** → Source: `main`, root folder
4. Live at `https://<your-username>.github.io/heyoka-enterprises/`

No build step needed — pure HTML + CSS + JS.
